To run the file with bird not loaded (initial version that shows how the bird learns):

python flappy_init.py

To run the file with pipes of fixed height (Grade 3 requirement):
python flappy_Fixed_Height.py (make sure that the file fixed_pipe.txt is present in the directory)

To run the file with pipes of different heights and same sequence (Grade 4 requirement):
python flappy_random.py  (make sure that the file random.txt is present in the directory)

To run the file with pipes of random sequence and height :
python random_try.py     (make sure that the file complete_random.txt is present in the directory)
